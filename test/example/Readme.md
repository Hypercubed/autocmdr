# Description

A autocmdr CLI app

# Usage

```
.bin/example --help
```

## auto-completion

Do one of the following to enable auto-completion in your shell.

* Add completion helper to ~/.bashrc (or ~/.zshrc) `example completion >> ~/.bashrc`
* Add completion to current shell `. <(example completion)`
